# DemoCodeFiled

Install and use:

```sh
git clone https://github.com/retyui/react-native-confirmation-code-field.git

cd react-native-confirmation-code-field/examples/DemoCodeFiled

# insall deps
yarn

# run metro bundler
yarn start

yarn run-android
# or
yarn run-ios

```
